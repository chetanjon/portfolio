# Hero sloth animation

The hero shows a small sloth that hangs on the headline line and swings between
grips. By default it's a lightweight hand-drawn SVG (in `components/sections/HeroSloth.tsx`).

## Upgrade to a premium animated (Lottie) sloth

Drop a Lottie file here as `sloth.json` and the component loads it automatically
(the SVG is the fallback):

1. Find a sloth animation you like on https://lottiefiles.com (or
   https://iconscout.com). Most free ones are free for commercial use **with
   attribution** — confirm the license on the asset's page.
2. Download the **Lottie JSON** and save it as `public/lottie/sloth.json`.
3. If the license requires attribution, record it below.

> No Lottie was bundled by default: the freely-downloadable sloth animations are
> account-gated and attribution-licensed (not public domain), so the choice +
> license is left to you rather than shipping an asset with murky licensing.

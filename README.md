# Chetan Jonnalagadda Portfolio

Personal portfolio for Chetan Jonnalagadda, focused on product case studies, projects, and experience.

## Tech Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```

App runs at `http://localhost:3000`.

## Scripts

- `npm run dev` - Start local dev server
- `npm run build` - Create production build
- `npm run start` - Run production build locally
- `npm run lint` - Run ESLint checks

## Project Structure

- `app/` - Routes, layouts, metadata, sitemap
- `components/` - Reusable UI/layout/section components
- `data/` - Structured content for portfolio sections
- `lib/` - Constants and shared utilities
- `public/` - Static assets and icons

## Content Updates

- Update navigation and global site details in `lib/constants.ts`.
- Update work/project content in the files under `data/`.
- Update case study pages in `app/casestudies/`.

## SEO and Metadata

- Root metadata is configured in `app/layout.tsx`.
- Sitemap is generated by `app/sitemap.ts`.
- Keep canonical domain aligned across metadata and constants.

## Deployment

Deploy to Vercel (recommended for Next.js):

1. Connect the repository in Vercel.
2. Set any required environment variables.
3. Deploy and verify metadata + sitemap in production.
